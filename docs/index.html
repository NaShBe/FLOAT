Welcome to FLOAT

About

FLOAT is a simple, open-source IEEE 754 Floating Point simulator and calculator. It is designed to help programmers, developers, and others interested in computer data understand how decimal values are represented and manipulated in computers via floating point numbers.

Try It Out

Press "Launch FLOAT" to use the web application. There is also a downloadable version available for offline use.

Read The Code

All source code for FLOAT can be found on GitHub.
